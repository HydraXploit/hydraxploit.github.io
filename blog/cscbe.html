Cyber Security Challenge Belgium 2025

On March 14–15, 2025, I participated in the Cyber Security Challenge Belgium (CSCBE) with Team Avalanche.

This nationwide event tests students in Capture-The-Flag (CTF) style cybersecurity scenarios.

Qualifiers included reverse engineering, cryptography, forensics, web exploitation, and trivia.

We placed 133rd. While not finalists, it was a valuable learning experience — especially under pressure.

This event reinforced my interest in real-world problem solving and team-based cybersecurity challenges.
