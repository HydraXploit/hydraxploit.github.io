Making This Site

I built this site from scratch using HTML, CSS, and a bit of JavaScript. No frameworks, no generators — just clean files that I control fully.

It's hosted using GitHub Pages, which means no servers or backend are required. Everything is static, which keeps it fast and simple to maintain.

Each post lives in its own HTML file, making it easy to add or change content without relying on plugins or external platforms.

The styling uses a dark, neon-accented theme inspired by cybersecurity culture. It’s clean, direct, and designed to reflect how I think about the field.

This portfolio is meant to grow with me. As I explore more tools and techniques, I’ll evolve the design and content to reflect that.
